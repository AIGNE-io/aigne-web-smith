# 使用特殊元件

AIGNE WebSmith 包含一個特殊元件庫，旨在以最少的設定為您的網站添加強大的視覺和互動元素。這些預先建構的元件處理複雜的功能，讓您能高效地創造引人入勝的使用者體驗。

本節概述了可用的特殊元件，並引導您至詳細的實作指南。透過利用這些工具，您可以顯著增強網頁的呈現方式和互動性。

## 可用元件

WebSmith 開箱即用地提供了幾個進階元件。主要元件詳述如下。有關具體的實作說明，請參閱連結的子頁面。

<x-cards data-columns="1">
  <x-card data-title="終端機播放器" data-icon="lucide:terminal" data-href="/advanced-features/use-special-components/terminal-player">
    一個用於將動畫化的終端機對話錄製直接嵌入到您的網頁中的元件。它非常適合用於教學、命令列工具示範和技術文件。
  </x-card>
  <x-card data-title="最新更新" data-icon="lucide:newspaper" data-href="/advanced-features/use-special-components/latest-updates">
    一個用於在您的網站上展示近期變更、新聞或部落格文章的元件。
  </x-card>
</x-cards>

使用特殊元件可以顯著提升您網站的視覺吸引力和功能。

有關如何使用這些元件的詳細說明，請參閱其各自的文件部分。
