# 使用特殊组件

AIGNE WebSmith 包含一个特殊组件库，旨在以最少的配置为您的网站添加强大的视觉和交互元素。这些预构建的组件处理复杂的功能，让您能够高效地创建引人入胜的用户体验。

本节概述了可用的特殊组件，并引导您查看实现这些组件的详细指南。通过利用这些工具，您可以显著增强网页的呈现效果和交互性。

## 可用组件

WebSmith 开箱即用地提供了几种高级组件。主要组件详述如下。有关具体的实现说明，请参阅链接的子页面。

<x-cards data-columns="1">
  <x-card data-title="终端播放器" data-icon="lucide:terminal" data-href="/advanced-features/use-special-components/terminal-player">
    一个用于将动画终端会话录像直接嵌入到网页中的组件。它非常适合教程、命令行工具演示和技术文档。
  </x-card>
  <x-card data-title="最新动态" data-icon="lucide:newspaper" data-href="/advanced-features/use-special-components/latest-updates">
    一个用于在您的网站上展示最近变更、新闻或博客文章的组件。
  </x-card>
</x-cards>

## 总结

使用特殊组件可以极大地提升您网站的视觉吸引力和功能。

有关如何使用这些组件的详细说明，请参阅它们各自的文档部分。
