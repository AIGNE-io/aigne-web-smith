# 高级功能

本节探讨 AIGNE WebSmith 的高级功能，这些功能允许在核心网站生成任务之外实现更强的定制性和功能性。一旦您熟悉了基础知识，就可以使用这些功能来微调您网站的构建模块，并融入专业的、具有视觉吸引力的元素。

此处介绍的功能主要集中在两个方面：管理 AI 用于构建页面的底层组件库，以及利用强大的预构建特殊组件来满足独特的展示需求。理解这些概念能让您更精确、更有创意地维护和增强您的网站。
