# インストール

AIGNE WebSmith でウェブサイトの生成を開始する前に、まず AIGNE コマンドラインインターフェース（CLI）をインストールする必要があります。このツールは、WebSmith フレームワークと対話するために必要なコマンドを提供します。このセクションでは、インストールプロセスのステップバイステップガイドを提供します。

## システム要件

*   **Node.js >=20**: AIGNE WebSmith には Node.js バージョン 20 以降が必要です
*   **npm**: npm (Node Package Manager) は Node.js と一緒に自動的にインストールされます

Node.js がインストールされていない場合は、[Node.js の公式サイト](https://nodejs.org/)からダウンロードしてインストールしてください。

## インストール

ターミナルで以下のコマンドを実行してください：

```bash
npm install -g @aigne/cli
```

## 高度なインストールオプション

### ベータ版

最新の機能をテストするためにベータ版をインストールするには：

```bash
npm install -g @aigne/cli@beta
aigne web upgrade --beta
```

### ローカルインストール

CLI をプロジェクトの依存関係としてインストールするには：

```bash
npm install @aigne/cli
```

ローカルにインストールした場合、コマンドの実行には `npx aigne ...` を使用します。

## 次のステップ

インストールが完了したら、最初のウェブサイトを作成する準備が整いました。次のセクションに進んで開始してください。
