# 入門指南

本指南提供了讓您快速上手 AIGNE WebSmith 的基本步驟。遵循這些說明，您將能在 30 分鐘內完成從安裝到產生並檢視您的第一個完整網站。整個過程分為三個合乎邏輯的階段，以確保設定過程清晰高效。

為了幫助您了解整個流程，以下是您將採取的步驟的簡單流程圖：
