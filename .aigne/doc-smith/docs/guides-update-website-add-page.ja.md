AIGNE WebSmithを使えば、ウェブサイトの拡張は簡単です。このガイドでは、`aigne web add-page`コマンドを使用して新しいページをサイト構造にシームレスに統合し、コンテンツと内部リンクを自動的に更新する方法を説明します。

# ページの追加

`aigne web add-page`コマンドは、ウェブサイトを拡張するための対話的な方法を提供します。これにより、新しいページを自然言語で記述するだけで、WebSmithがページの作成、コンテンツの生成、そして既存ページへの関連リンクのインテリジェントな更新プロセスを処理します。

## 仕組み

コマンドを実行すると、WebSmithは対話型セッションを開始します。

1.  **現在の構造を表示**: コマンドはまず現在のサイトマップを出力し、既存ページの明確な概要を把握できるようにします。
2.  **対話型プロンプト**: 追加したいページについて記述するよう促されます。タイトル、構造内の場所、目的（例：「メインのサービスページの下に価格ページを追加」）を指定できます。
3.  **反復的な追加**: 1つのセッションで複数のページを追加できます。各ページを追加した後、ツールは更新された構造を表示し、次のページを尋ねます。終了するには、空のプロンプトでEnterキーを押すだけです。
4.  **コンテンツ生成**: 構造を確定すると、WebSmithはプロジェクトのソース資料に基づいて各新しいページのコンテンツを生成します。
5.  **自動リンク**: ツールは既存のコンテンツを分析し、新しいページへのリンクを挿入する関連箇所を特定し、ユーザーにとってシームレスなナビゲーションを保証します。
6.  **翻訳と更新**: サイトが多言語を使用している場合、新しいページと更新されたリンクは自動的に翻訳ワークフローに含まれます。
7.  **サマリーレポート**: 最後に、作成された新しいページと、新しいリンクで更新された既存ページのリストを含むサマリーが表示されます。

## 使用方法

ページを追加するには、プロジェクトのルートディレクトリに移動し、次のコマンドを実行します。

```sh aigne web add-page icon=lucide:terminal
aigne web add-page
```

WebSmithは現在のウェブサイト構造を表示し、追加したい新しいページを指定するように促します。

![コマンドラインインターフェースがユーザーに新しいページの追加を促します。](../../../assets/images/web-smith-add-page-input.png)

例えば、次のように入力できます。

```
add a 'Team' page under '/about'
```

WebSmithはこのリクエストを処理し、構造を更新して結果を表示します。既に存在するパスでページを追加しようとすると、システムは通知し、別のパスを要求します。

![CLIはページパスがすでに存在するという警告を表示します。](../../../assets/images/web-smith-add-page-success.png)

さらにページを追加し続けるか、Enterキーを押して変更を確定し、コンテンツ生成プロセスを開始することができます。

## まとめ

`aigne web add-page`コマンドは、ウェブサイトを拡張するプロセスを簡素化します。対話的な構造管理と自動化されたコンテンツ生成およびリンク統合を組み合わせることで、サイトが成長しても一貫性があり、よく整理された状態を保ちます。

関連する操作については、[ページの削除](./guides-update-website-remove-page.md)の方法を参照するか、[ページの更新](./guides-update-website-update-page.md)コマンドでより一般的なコンテンツの改良を行ってください。
