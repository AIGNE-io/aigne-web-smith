使用 AIGNE WebSmith 來擴展您的網站非常簡單。本指南將說明如何使用 `aigne web add-page` 指令，將新頁面無縫整合至您的網站結構中，並自動更新內容和內部連結。

# 新增頁面

`aigne web add-page` 指令提供了一種互動式的方式來擴展您的網站。它允許您使用自然語言描述新頁面，WebSmith 會處理建立頁面、產生內容，並智慧地更新現有頁面以加入相關的新內容連結。

## 運作方式

當您執行此指令時，WebSmith 會啟動一個互動式會話：

1.  **顯示目前結構**：指令會先印出目前的網站地圖，讓您清楚了解現有頁面。
2.  **互動式提示**：系統會提示您描述要新增的頁面。您可以指定標題、在結構中的位置及其用途（例如，「在主要服務頁面下新增一個定價頁面」）。
3.  **迭代式新增**：您可以在單一會話中新增多個頁面。每次新增後，工具會顯示更新後的結構，並提示您輸入下一個頁面。若要完成，只需在空白提示符處按下 Enter 鍵。
4.  **內容產生**：一旦您確認結構，WebSmith 會根據您專案的來源資料為每個新頁面產生內容。
5.  **自動連結**：工具會分析您現有的內容，找出相關位置插入新頁面的連結，確保使用者能無縫導覽。
6.  **翻譯與更新**：如果您的網站使用多種語言，新頁面和更新後的連結會自動納入翻譯工作流程中。
7.  **摘要報告**：最後，會顯示一份摘要，列出已建立的新頁面以及任何已更新並加入新連結的現有頁面。

## 使用方法

若要開始新增頁面，請導覽至您專案的根目錄並執行以下指令：

```sh aigne web add-page icon=lucide:terminal
aigne web add-page
```

WebSmith 將顯示您目前的網站結構，並提示您指定要新增的頁面。

![命令列介面提示使用者新增一個新頁面。](../../../assets/images/web-smith-add-page-input.png)

例如，您可以輸入：

```
add a 'Team' page under '/about'
```

WebSmith 將處理此請求，更新結構，並顯示結果。如果您嘗試新增一個路徑已存在的頁面，系統會通知您並提示您輸入不同的路徑。

![CLI 顯示一個頁面路徑已存在的警告。](../../../assets/images/web-smith-add-page-success.png)

您可以繼續新增更多頁面，或按下 Enter 鍵以確認變更並開始內容產生過程。

## 摘要

`aigne web add-page` 指令簡化了擴展網站的流程。它結合了互動式結構管理與自動化內容生成和連結整合，確保您的網站在成長過程中保持一致且井然有序。

如需相關操作，請參閱如何[移除頁面](./guides-update-website-remove-page.md)或使用[更新頁面](./guides-update-website-update-page.md)指令進行更一般的內容調整。
