使用 AIGNE WebSmith 扩展您的网站非常简单。本指南将说明如何使用 `aigne web add-page` 命令将新页面无缝集成到您的网站结构中，并自动更新内容和内部链接。

# 添加页面

`aigne web add-page` 命令提供了一种交互式的方式来扩展您的网站。它允许您用自然语言描述新页面，WebSmith 会负责创建页面、生成内容，并智能地更新现有页面，添加指向新内容的相关链接。

## 工作原理

当您运行该命令时，WebSmith 会启动一个交互式会话：

1.  **显示当前结构**：该命令首先会打印出当前的站点地图，以便您清晰地了解现有页面。
2.  **交互式提示**：系统会提示您描述希望添加的页面。您可以指定标题、其在结构中的位置以及其用途（例如，“在主服务页面下添加一个定价页面”）。
3.  **迭代添加**：您可以在单个会话中添加多个页面。每次添加后，工具都会显示更新后的结构，并提示您输入下一个页面。在空提示符下直接按 Enter 键即可完成操作。
4.  **内容生成**：当您最终确定结构后，WebSmith 会根据您项目的源材料为每个新页面生成内容。
5.  **自动链接**：该工具会分析您现有的内容，以确定在何处插入指向新页面的相关链接，从而确保用户能够无缝导航。
6.  **翻译与更新**：如果您的网站使用多种语言，新页面和更新后的链接将自动包含在翻译工作流中。
7.  **摘要报告**：最后，系统会显示一份摘要，列出已创建的新页面以及所有已更新并添加了新链接的现有页面。

## 用法

要开始添加页面，请导航到您项目的根目录并运行以下命令：

```sh aigne web add-page icon=lucide:terminal
aigne web add-page
```

WebSmith 将显示您当前的网站结构，并提示您指定要添加的新页面。

![命令行界面提示用户添加新页面。](../../../assets/images/web-smith-add-page-input.png)

例如，您可以输入：

```
add a 'Team' page under '/about'
```

WebSmith 将处理此请求，更新结构，并显示结果。如果您尝试添加的页面路径已存在，系统会通知您并提示您使用其他路径。

![CLI 显示一个页面路径已存在的警告。](../../../assets/images/web-smith-add-page-success.png)

您可以继续添加更多页面，或按 Enter 键确认更改并开始内容生成过程。

## 摘要

`aigne web add-page` 命令简化了扩展网站的过程。它将交互式结构管理与自动化内容生成和链接集成相结合，确保您的网站在成长过程中保持一致性和良好的组织性。

有关相关操作，请参阅如何[移除页面](./guides-update-website-remove-page.md)或使用 [更新页面](./guides-update-website-update-page.md) 命令进行更常规的内容优化。
