# 更新網站

使用本指南選擇正確的更新工作流程，然後深入了解有關結構或頁面級別變更的詳細子指南。

網站生成後，您將不可避免地需要進行變更。AIGNE WebSmith 提供了一種清晰且結構化的方法來保持您的網站最新。更新通常分為兩類：修改網站的整體結構或優化單個頁面中的內容。

本節對這兩個過程進行了高層次的概述。有關詳細的逐步說明，請參閱下面連結的特定子文件。

## 總結

通過將結構調整、頁面級別的優化和審核追蹤分開，WebSmith 提供了一個邏輯且高效的工作流程來管理您的網站。

完成您想要的變更後，下一步通常是將其上線。有關此方面的指導，請參閱 [發佈網站](./guides-publish-website.md) 一文。
