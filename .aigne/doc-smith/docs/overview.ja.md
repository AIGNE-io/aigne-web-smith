# 概要

このドキュメントでは、AIGNE WebSmith の概要を説明します。読み終える頃には、その中心的な目的、主要な機能、そして初期計画から最終的な公開までプロフェッショナルなウェブサイトの作成を自動化するために使用する基本的なワークフローを理解できるでしょう。

AIGNE WebSmith は、プロフェッショナルで SEO に最適化されたウェブサイトの作成を自動化する AI 駆動のツールです。提供されたソース資料に基づいて完全なコンテンツとテンプレートを生成し、完成したサイトを直接公開できます。このシステムは、ウェブ開発プロセス全体を合理化し、手作業や技術的な要件を最小限に抑えるように設計されています。

## 主な機能

WebSmith は、ウェブサイトの自動作成と管理のための包括的な機能セットを提供します。このシステムは、AI 駆動の生成、プロフェッショナルなテンプレートシステム、品質保証、そして直接公開という 4 つの主要な機能を中心に構成されています。

| 機能領域 | 説明 |
| :--- | :--- |
| **AI による生成** | AI を活用して要件を分析し、最適なサイト構造を計画し、一度の操作ですべてのページの詳細なコンテンツを生成し、SEO のベストプラクティスを適用します。英語と中国語の両方のコンテンツ生成をサポートしています。 |
| **プロフェッショナルなテンプレート** | Pages Kit と互換性のあるテンプレートを作成し、Hero、CTA、FAQ などの要素を持つコンポーネントベースのデザインを使用します。テンプレートはレスポンシブで、モバイルとデスクトップの両方のディスプレイに適応します。 |
| **品質保証** | ウェブサイトのアーキテクチャ、ユーザーエクスペリエンス、コンテンツの一貫性を評価するための自動評価システムが含まれています。コア機能をカバーする 32 以上のテストケース群によってサポートされています。 |
| **ワンクリック公開** | 複数ページのウェブサイトを Pages Kit に直接一括公開できます。ステータス監視機能を提供し、デプロイが成功するとすぐに公開アクセスリンクを提供します。 |

## 仕組み

WebSmith は、ウェブサイトを構築しデプロイするために順次動作する一連の特化した AI Agent を採用しています。各 Agent はプロセスの特定の段階を担当し、構想から公開まで構造化された信頼性の高いワークフローを保証します。

プロセスは以下の通りです。

1.  **構造計画**: Agent がプロジェクト要件を分析し、インテリジェントで論理的なウェブサイトアーキテクチャを設計します。
2.  **コンテンツ生成**: 計画された構造に基づき、別の Agent が各ページに対して高品質で関連性の高いコンテンツを生成します。
3.  **テンプレート生成**: 3 番目の Agent が生成されたコンテンツを受け取り、プロフェッショナルなコンポーネントベースのテンプレートに組み立てます。
4.  **評価と公開**: 最後に、評価 Agent が出力の品質を評価した後、公開 Agent が完成したウェブサイトをアップロードします。

このモジュール式の Agent ベースのアーキテクチャにより、各ステップが専門的なプロセスによって処理され、一貫性のある高品質な最終製品が保証されます。
