# Reference

This section provides detailed technical material for AIGNE WebSmith. It is designed for users seeking specific, in-depth information about commands, configuration options, material preparation, and licensing. For practical, step-by-step guides, please consult the [Getting Started](./getting-started.md) or [Guides](./guides.md) sections.
